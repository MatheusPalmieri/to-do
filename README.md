### Rodando A Aplicação

Clone o projeto com

```sh
git clone https://github.com/MatheusPalmieri/to-do
```

Entre na pasta do projeto e instale as dependências com:

```sh
npm i -g expo-cli
```

Execute o comando para inicializar a aplicação

```sh
expo start
```
ou

```sh
npm start
```
